# 🌌 Pathik Patel | Full-Stack Architect Portfolio

A world-class, futuristic personal portfolio designed with a focus on **Glassmorphism**, high-end aesthetics, and professional identity. Built for performance and creativity using pure **Vanilla JavaScript, CSS3, and HTML5**.

---

## ✨ Design Philosophy
This portfolio is built on the **"Submerged" Design System**, featuring:
- **Glassmorphism UI**: High-blur, translucent cards with neon accents.
- **Dynamic Atmosphere**: A mouse-following radial blob and a node-link background canvas.
- **Micro-Interactions**: Smooth scroll reveals, state-aware counters, and refined transition curves.
- **High-End Typography**: Utilizing *Outfit* for futuristic headers and *Plus Jakarta Sans* for modern readability.

## 🚀 Technical Features
- **Zero Dependencies**: 100% Vanilla JS/CSS. No frameworks, no bloat.
- **Content Architect (Admin Mode)**: A built-in editor that allows you to live-add skills, projects, and experiences directly from the UI.
- **Persistent Storage**: Uses `LocalStorage` to save your custom additions across browser sessions.
- **Performance Optimized**: SVG noise filters and optimized Canvas API for fluid animations at 60fps.
- **Fully Responsive**: Mobile-first architecture with custom breakpoints for all screens.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Modern Flex/Grid, Custom Vars).
- **Backend Logic**: Vanilla JavaScript (ES6+).
- **Icons & Fonts**: Google Fonts, FontAwesome 6.0.
- **Deployment**: Optimized for GitHub Pages.

## ✍️ How to Use the Content Architect
1. **Open the Site**: Launch `index.html`.
2. **Access Admin**: Click the **Pen Nib Icon** in the top-right corner of the navigation bar.
3. **Add Content**:
   - **Skills**: Add new categories and comma-separated skill lists.
   - **Projects**: Input titles, descriptions, and tags.
   - **Experience**: Log your career journey with bullet points.
4. **Save**: Click the "Add" button, and the site will instantly re-render and persist your data.
